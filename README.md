# Movie List — Список фильмов

Минималистичное веб-приложение для ведения личного списка фильмов к просмотру. Позволяет добавлять фильмы, отмечать просмотренные и удалять записи — без регистрации и лишних действий.

## ✨ Функциональность

- **Добавление фильма** — вводишь название в поле и нажимаешь кнопку, фильм появляется вверху списка
- **Отметка о просмотре** — чекбокс рядом с названием помечает фильм как просмотренный (визуально выделяется)
- **Удаление** — кнопка удаления убирает фильм из списка
- **Событийная делегация** — все клики по списку обрабатываются через один слушатель на родительском элементе

> ⚠️ Данные хранятся только в памяти браузера и сбрасываются при перезагрузке страницы.

## 🛠️ Стек

| Технология | Описание |
|---|---|
| HTML5 | Разметка страницы |
| CSS3 | Стили и оформление |
| JavaScript (ES6+) | Логика приложения |

Без фреймворков и сборщиков — чистый vanilla JS, запускается напрямую в браузере.
